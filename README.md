# PyQt5 Calculator

A professional GUI calculator built with Python and PyQt5 for college assignment submission.

## Features

- Basic arithmetic: addition, subtraction, multiplication, division
- Decimal numbers and percentage support
- Continuous calculations (chain results with more operators)
- Input validation (no double decimals, no invalid operator chains)
- Error handling for division by zero and invalid expressions
- Dark / light theme toggle
- Full keyboard support

## Requirements

- Python 3.7 or newer
- PyQt5

## Installation

Open a terminal in this project folder and run:

```
pip install PyQt5
```

## How to Run

```
python calculator.py
```

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0`–`9` | Enter digits |
| `+` `-` `*` `/` | Operators |
| `.` | Decimal point |
| `%` | Percentage |
| `Enter` | Calculate (`=`) |
| `Backspace` / `Delete` | Delete last character |
| `Escape` | All clear (`AC`) |
